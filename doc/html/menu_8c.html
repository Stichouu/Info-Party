la fonction permettant d'afficher notre menu en conservant la fenêtre et permettant d'accéder à la création d'une partie,de rejoindre une partie,des options ou à quitter le jeu en cliquant sur la case dédiée.

Paramètres

tourPointeur sur un objet de type integer pour changer si besoin sa valeur pour avoir le nombre de tours.
quitterPointeur sur un objet de type integer pour changer si besoin sa valeur afin de savoir si on quitte le programme.
